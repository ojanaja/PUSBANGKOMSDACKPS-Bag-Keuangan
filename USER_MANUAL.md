## 1. Pendahuluan
SiAP-PUSBANGKOMSDACKPS adalah sistem informasi yang dikembangkan untuk membantu proses pencairan dana dan manajemen keuangan di lingkungan PUSBANGKOMSDACKPS. Sistem ini mendigitalkan proses pengajuan, verifikasi, hingga persetujuan anggaran dan paket pekerjaan.

---

## 2. Akses Sistem

### Login
1. Buka browser (Chrome, Firefox, Safari, atau Edge disarankan).
2. Masukkan alamat URL aplikasi (Contoh: `http://localhost` atau domain yang disediakan admin).
3. Halaman Login akan muncul.
4. Masukkan **NIP/Username** dan **Password** Anda.
5. Klik tombol **Masuk**.

> **Catatan:** Jika Anda lupa password, silakan hubungi Administrator Sistem.

---

## 3. Fitur Utama

### 📊 Dashboard
Setelah berhasil login, Anda akan diarahkan ke halaman Dashboard. Halaman ini menampilkan ringkasan penting:
- Status realisasi anggaran.
- Grafik progess penyerapan.
- Notifikasi tugas yang tertunda (pending tasks).

### 💰 Manajemen Anggaran
Menu ini digunakan untuk melihat dan mengelola data DIPA/Anggaran.
- **Import Anggaran**: Fitur untuk mengupload data anggaran baru (biasanya format Excel/CSV standar Kemenkeu/SAKTI).
- **Monitoring**: Melihat sisa pagu anggaran per program/kegiatan.

### 📦 Manajemen Paket
Inti dari aplikasi ini adalah pengelolaan paket pengadaan/pencairan.
1. **Buat Paket Baru**: Isi formulir detail paket (Nama Paket, Nilai, Sumber Dana).
2. **Upload Dokumen**: Unggah dokumen pendukung (Kwitansi, SK, dll).
3. **Tracking Status**: Pantau status paket apakah `Draft`, `Diajukan`, `Diverifikasi`, atau `Disetujui`.

### ✅ Approval & Audit
Untuk pengguna dengan role Verifikator atau PPK:
- **Daftar Tugas**: Lihat daftar paket yang menunggu persetujuan Anda.
- **Review Dokumen**: Periksa kelengkapan dokumen secara online tanpa perlu berkas fisik.
- **Keputusan**: Klik **Setuju** untuk melanjutkan proses atau **Tolak** (dengan catatan perbaikan) jika ada kekurangan.

### 🔔 Notifikasi
Sistem dilengkapi dengan fitur Early Warning System (EWS) dan notifikasi.
- Ikon lonceng di pojok kanan atas menampilkan pemberitahuan terbaru.
- Notifikasi akan muncul jika:
    - Ada paket baru yang perlu diverifikasi.
    - Ada paket Anda yang disetujui/ditolak.
    - Batas waktu pencairan sudah dekat.

## 4. FAQ & Troubleshooting

**Q: Saya tidak bisa login, muncul pesan "Invalid Credentials".**
A: Periksa kembali NIP dan Password Anda. Pastikan Caps Lock tidak aktif. Jika masih gagal, hubungi Admin IT.

**Q: Dokumen apa saja yang bisa diupload?**
A: Sistem umumnya menerima format PDF, JPG, dan PNG. Ukuran maksimal file biasanya dibatasi (misal: 10MB per file).

**Q: Bagaimana cara mengganti password?**
A: Klik profil Anda di pojok kanan atas, pilih menu "Pengaturan" atau "Ganti Password", lalu ikuti instruksi di layar.

**Q: Halaman tidak memuat data (Loading terus-menerus).**
A: Coba refresh halaman (F5). Jika masalah berlanjut, periksa koneksi internet Anda. Jika internet lancar namun aplikasi tetap error, laporkan ke tim teknis.
